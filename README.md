# Programming Logic Course

This repository contains projects and exercises from a programming logic course. The goal is to learn new languages, practice programming fundamentals, and improve problem-solving skills.

## Structure

- `linguagens/`: Contains language-specific examples and exercises.
  - `C/`: Basic programs and exercises in C.
  - `C#/`: Basic programs and exercises in C#.
  - `C++/`: Basic programs and exercises in C++.
  - `Java/`: Basic programs and exercises in Java.
  - `Python/`: Basic programs and exercises in Python.

## Contents

Each language folder includes:
- basic syntax examples
- input/output examples
- control structures
- collections and matrix handling
- simple exercises like age calculation, area, sums, and conditional logic

## How to use

1. Choose a language folder.
2. Open the example or exercise file.
3. Compile/run the program using the appropriate language tools:
   - C: `gcc` or another C compiler
   - C#: `csc` or .NET SDK
   - C++: `g++` or another C++ compiler
   - Java: `javac` and `java`
   - Python: `python` or `python3`

## Notes

This repository is intended for learning and practicing core programming concepts across multiple languages.
